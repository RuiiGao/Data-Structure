## 二叉树的遍历
- 二叉树递归先根遍历的算法
  - 思路：
    - 若二叉树为空，则空操作
    - 否则：
      - 访问根结点
      - 先根遍历左子树
      - 先根遍历右子树
  - 参考题目：[LeetCode 144 二叉树的前序遍历](https://leetcode.cn/problems/binary-tree-preorder-traversal/description/)
  - 代码实现：
   ```cpp {.line-numbers}
   void visit(int data){
    printf("%d ", data);
   }

   void Preorder(TreeNode* root){
    if(root == NULL) return;
    visit(root->data);
    Preorder(root->left);
    Preorder(root->right);
   }
   ```
  - 复杂度：
    - 时间复杂度 `O(n)` 、空间复杂度 `O(h)`
    - 其中 `n` 为二叉树结点数，`h` 为二叉树高度
  - 蕴含思想之**分治法**与**动态规划**的比较：
    - **分治法**：
      - 子问题相互独立不重叠
      - 自顶向下递归实现
    - **动态规划**：
      - 子问题间往往具有重叠性，可将子问题的解存入表中，以后再遇到相同的子问题直接查表
      - 自底向上递推实现
- 二叉树中根遍历的递归算法
  - 思路：
    - 若二叉树为空，则空操作
    - 否则：
      - 中根遍历左子树
      - 访问根结点
      - 中根遍历右子树
  - 参考题目：[LeetCode 94 二叉树的中序遍历](https://leetcode.cn/problems/binary-tree-inorder-traversal/description/)
  - 代码实现：
   ```cpp {.line-numbers}
   void visit(int data){
    printf("%d ", data);
   }

   void Inorder(TreeNode* root){
    if(root == NULL) return;
    Inorder(root->left);
    visit(root->data);
    Inorder(root->right);
   }
   ```
  - 复杂度：
    - 时间复杂度 `O(n)` 、空间复杂度 `O(h)`
    - 其中 `n` 为二叉树结点数，`h` 为二叉树高度
- 二叉树后根遍历的递归算法
  - 思路：
    - 若二叉树为空，则空操作
    - 否则：
      - 后根遍历左子树
      - 后根遍历右子树
      - 访问根结点
    - 参考题目：[LeetCode 145 二叉树的后序遍历](https://leetcode.cn/problems/binary-tree-postorder-traversal/description/)
    - 代码实现：
   ```cpp {.line-numbers}
   void visit(int data){
    printf("%d ", data);
   }

   void Postorder(TreeNode* root){
    if(root == NULL) return;
    Postorder(root->left);
    Postorder(root->right);
    visit(root->data);
   }
   ```
  - 复杂度：
    - 时间复杂度 `O(n)` 、空间复杂度 `O(h)`
    - 其中 `n` 为二叉树结点数，`h` 为二叉树高度
