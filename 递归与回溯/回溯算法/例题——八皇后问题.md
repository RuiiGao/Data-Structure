## 题目说明
- 说明：8 * 8棋盘，如何放置8个皇后，使得任意两个皇后都不在同一行、同一列、同一斜线
- 参考题目：[HDU 2553 N皇后问题](https://acm.hdu.edu.cn/showproblem.php?pid=2553)
- **解题核心**：`ans[k]` 表示第 k 列的皇后放到第几行

## 穷（枚）举法
算法说明：
```cpp {.line-numbers}
算法 Nqueens(ans)
    for(ans[1] 从 1 取到 8)
        for(ans[2] 从 1 取到 8)
            for(ans[3] 从 1 取到 8)
                for(ans[4] 从 1 取到 8)
                    for(ans[5] 从 1 取到 8)
                        for(ans[6] 从 1 取到 8)
                            for(ans[7] 从 1 取到 8)
                                for(ans[8] 从 1 取到 8)
                                    if(feasible(ans))
                                        print(ans);
```

## 回溯法
- 递归代码：
  ```cpp {.line-numbers}
    void Nqueens(int ans[], int n, int k){ //填解数组的第k位ans[k]
        if(k > n) {print(ans, n); return;} // 解数组的 n 位都填完了
        for(int i = 1; i <= n; i++){
            if(feasible(ans, k, i)){ // ans[k] 是否可以等于 i
                ans[k] = i;
                Nqueens(ans, n, k + 1);
            }
        }
    }
  ```
  - 初始调用 `Nqueens(ans,n,1)`
- 可行性判断代码：
  ```cpp {.line-numbers}
    bool feasible(int ans[], int k, int i){
        // ans[k] 是否可以等于 i ，即第 k 列第 i 行是否可以放置皇后
        for(int j = 1; j <= k - 1; j++)
            if(ans[j] == i || abs(ans[j] - i) == abs(j - k)) 
                return false;
            return true;
    }
  ```
