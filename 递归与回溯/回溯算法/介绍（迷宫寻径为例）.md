## 引入——迷宫寻径
- 题目说明：
  - 给定 `n * m` 的迷宫，从起始位置 `(sx, sy)` 开始每步走一格，可移动至上、下、左、右任意一个相邻的格子，不能斜着走，有障碍物的格子不能走
  - 如何能走到目标位置 `(tx, ty)` ？输出所有可能的方案（保证起点和终点没有障碍物）
  - 参考题目：
    - [洛谷 P1605 迷宫](https://www.luogu.com.cn/problem/P1605)
    - [洛谷 B3625 迷宫寻路](https://www.luogu.com.cn/problem/B3625)

### 具体分析
- 相关动作说明：
  - 0 —— 上：`dx[0] = -1, dy[0] = 0`
  - 1 —— 右：`dx[1] = 0, dy[1] = 1`
  - 2 —— 下：`dx[2] = 1, dy[2] = 0`
  - 3 —— 左：`dx[3] = 0, dy[3] = -1`
  - `int dx[4] = {-1, 0, 1, 0}; int dy[4] = {0, 1, 0, -1};`
- 在 `(x, y)` 处执行动作 i，到达的新位置 `(nx, ny)` 为:
  - `nx = x + dx[i];`
  - `ny = y + dy[i];`
- **准备工作**的有关代码：
  ```cpp {.line-numbers}
    const int maxsize = 110;
    int map[maxsize][maxsize]; // 地图：空位 0，障碍 1
    int vis[maxsize][maxsize] = {{0}};
    // 用于标记走过的格子，规模与 map 相同：vis[i][j] = 0 表示点 (i,j) ，没被访问过，vis[i][j] = 1 表示点 (i,j) 被访问过
    int n, m; // 迷宫的规模 n 行 m 列

    bool feasible(int x, int y){ // 迷宫第 x 行第 y 列的格子能否走
        if(x >= 0 && x < n && y >= 0 && y < m && vis[x][y] == 0 && map[x][y] == 0)
            return true;
        return false;
    }

    void print(int ans[], int k){
        for(int i = 1; i < k; i++)
            printf("%d ", ans[i]);
        printf("\n");
    }
  ```
- **核心部分**代码：
  ```cpp {.line-numbers}
    void find(int x, int y, int tx, int ty, int ans[], int k){ // 已 (x,y) 为起点搜索，填解数组的第 k 位 ans[k]
        if(x == tx && y == ty){
            print(ans, k);  return;
        }
        vis[x][y] = 1; // 访问点(x,y)，终点的 vis 值始终不置为 1
        for(int i = 0; i < 4; i++){
            int nx = x + dx[i], ny = y + dy[i];
            if(feasible(nx, ny)){
                ans[k] = i;
                find(nx, ny, tx, ty, ans, k + 1);
                // 以 (nx, ny) 为起点继续搜索,并填 ans 数组的第 k+1 位
            }
        }
        vis[x][y] = 0; // 不在当前行进路线上的点 vis 值清0，保证只有在当前行进路线上的点的 vis 值为1，以便找到所有路线
    }
  ```
  - 注意初始调用为：`find(sx, sy, tx, ty, ans, 1)`
  - **回溯算法**核心：可行则进、不行则换、换不了则退
  - 关于函数返回前是否有必要把 `vis[x][y]` 清0：
    - vis 值清0：确保找到所有解
    - vis 值不清0：更快找到一个解
<br>

- 如果只找一个解，核心代码如下：
  ```cpp {.line-numbers}
    bool find(int x, int y, int tx, int ty, int ans[], int k){ //填解数组的第 k 位 ans[k] ，当前行进路线上的点的 visit 值置1
        if(x == tx && y == ty) {print(ans, k); return true;}
        vis[x][y] = 1;
        for(int i = 0; i < 4; i++){
            int nx = x + dx[i], ny = y + dy[i];
            if(feasible(nx, ny)){
                ans[k] = i;
                if(find(nx, ny, tx, ty, ans, k + 1)) return true;
            }
        }
        return false;
    }
  ```
  - 代码逻辑核心：只要找到一个解，就不断向上返回 `true`

## 回溯算法的基本框架
- 寻找所有方案：
  ```cpp {.line-numbers}
    算法 F(k) // 填解数组的第 k 位 ans[k]
        if(ans 已是一个解) {print(ans); return;} // 输出
        for(用 i 遍历 ans[k] 的所有可能的取值){
            if(feasible(i)){ // 如果 i 可行
                ans[k] = i;
                F(k + 1); // 填 ans[k + 1]
            }
        } #
  ```
- 只找一个解：
  ```cpp {.line-numbers}
    算法 F(k) // 填解数组的第 k 位 ans[k]
        if(ans 已是一个解) {print(ans); return true;}
        for(用 i 遍历 ans[k] 的所有可能的取值){
            if(feasible(i)){ // 如果 i 可行
                ans[k] = i;
                if(F(k + 1) == true) return true;
            }
        }
        return false; #
  ```
- 注意初始调用 F(1)
- **回溯**：可行则进，不行则换，换不了则退
