## 题目说明
- 给定实数 a 和正整数 n，编写算法求 $a ^ n$，不允许调用库函数
- 参考题目：[LeetCode 50 Pow(x, n)](https://leetcode.cn/problems/powx-n/description/)

## 硬解法
- 代码：
  ```cpp {.line-numbers}
    double pow(double a, long long n){
        double ans = 1;
        for(int i = 0; i < n; i++)
            ans *= a;
        return ans;
    }
  ```
- 时间复杂度：`O(n)`

## 快速幂算法
- 实现原理：
$$
a^n = 
\begin{cases}
a^{\frac{n}{2}} \times a^{\frac{n}{2}}, & n \text{ 是偶数} \\
a^{\left\lfloor \frac{n}{2} \right\rfloor} \times a^{\left\lfloor \frac{n}{2} \right\rfloor} \times a, & n \text{ 是奇数}
\end{cases}
$$
- 代码：
  ```cpp {.line-numbers}
    double pow(double a, long long n){
        if(a == 1 || n == 0) return 1;
        double b = pow(a, n / 2);
        double ans = b * b;
        if(n % 2 == 1) ans *= a;
        return ans;
    }
  ```
- 时间复杂度：`O(logn)`
