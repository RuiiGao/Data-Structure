## 介绍：
- 若矩阵 `A` 非零元素的个数远小于零元素的个数，则称 `A` 为稀疏矩阵
  - 压缩存储：仅存储非零元素，节省空间
  - 特点：零元素分布一般没有规律，难以通过公式实现压缩存储的地址映射
- 用三元组结点来存储矩阵的每个非零元素$a_{ij}$，其中 i 和 j 为元素的行号和列号：**| `i` | `j` | $a_{ij}$ |**
- 在三元组结点的基础上实现对整个稀疏矩阵的存储：
  - **顺序存储**方式实现：三元组表
  - **链接存储**方式实现：十字链表

## 三元组表
- **稀疏矩阵的转置算法**: 时间复杂度 `O(nt)`
  ```cpp {.line-numbers}  
    struct Triple{
        int row;
        int col;
        int value;
    };

    Triple a[100], b[100]; 

    void Transpose(Triple a[], int m, int n, int t, Triple b[]){
        // 将三元组表 a 表示的 m 行 n 列矩阵转置，存于三元组表 b 中， a 中非 0 元素个数为 t
        int j = 0; // j 标识当前填三元组 b 的第几位
        if(t == 0) return; // a 空
        for(int k = 1; k <= n; k++) // 填转置后的矩阵的第 k 行的元素
            for(int i = 0; i < t; i++) // 扫描矩阵 a ，看哪些元素列号是 k
                if(a[i].col == k){ //看 a 的哪些元素列号是 k
                    b[j].row = k; // 转置后行号应为 k
                    b[j].col = a[i].row; // 列号应为其在 a 中的行号
                    b[j].value = a[i].value;
                    j++; // 赋值三元组表 b 中的下一个结点
                }
    }
  ```
- **稀疏矩阵的快速转置算法**：时间复杂度 `O(n+t)`
  - 核心思路：
    - `start[1] = 0`
    - `start[2] = start[1] + 转置后第 1 行的元素个数`
    - `start[i] = start[i-1] + 转置后第 i-1 行的元素个数`
  - 代码说明：
    - `rowsize[ ]` ：长度为 `n` ，存放转置后各行非 0 元素的个数，即转置前各列非 0 元素的个数
     ```cpp {.line-numbers}
    for(int i = q; i <= n; i++)
        rowsize[i] = 0;
    for(int i = 0; i < t; i++){
        k = a[i].col;
        roesize[k]++;
    }
     ```
    - `start[ ]` ：长度为 `n` ，存放转置后的矩阵每行在三元组表 `b` 中起始位置
     ```cpp {.line-numbers}
    start[1] = 0;
    for(int i = 2; i <= n; i++)
        start[i] = start[i - 1] + rowsize[i - 1];
    for(int i = 0; i < t; i++){
        k = a[i].col;
        j = start[k];
        b[j].row = a[i].col;
        b[j].col = a[i].row;
        b[j].value = a[i].value;
        start[k]++;
    }
     ```
- 存储方式分析：
  - 节省空间，但对于非零元的位置或个数经常发生变化的矩阵运算就显得不太适合
  - 如：矩阵某些位置频繁的加上或减去一个数，使有的元素由 0 变成非 0 ，由非 0 变成 0。导致三元组表频繁进行插入删除操作，需要频繁元素移动

## 十字链表
- 介绍：
  - 矩阵的每一行、列都设置为由一个哨位结点引导的循环链表
  - `headRow[i]` 是第 `i` 行链表的头指针，指向第 `i` 行链表的哨位结点
  - `headCol[j]` 是第 `j` 列链表的头指针，指向第 `j` 列链表的哨位结点
  - 若第 `i` 行无非零元素，则 `headRow[i]->left == headRow[i]`
  - 若第 `j` 列无非零元素，则 `headCol[j]->up == headCol[j]`
  - 对矩阵的运算实质上就是在十字链表中插入结点、删除结点以及改变某个结点的数据域的值
- 拓展之**双向十字链表**之**舞蹈链**的应用：
  ```cpp {.line-numbers}
    // 移除一个元素
    p->left->right = p->right;
    p->right->left = p->left;
    p->up->down = p->down;
    p->down->up = p->up;
  ```
- 十字链表的类的封装实现：
  ```cpp {.line-numbers}
    #include <iostream>
    #include <vector>
    #include <cmath>
    #include <iomanip>

    using namespace std;

    // 定义十字链表的结点结构
    struct ListNode {
        int row; // 该结点所在行
        int col; // 该结点所在列
        double value; // 数据 
        ListNode* left; // 指向左侧相邻非零元素的指针 (列号递减方向)
        ListNode* up; // 指向上方相邻非零元素的指针 (行号递减方向)

        // 构造函数
        ListNode(int r, int c, double v) : row(r), col(c), value(v), left(nullptr), up(nullptr) {}
    };

    // 封装十字链表类
    class CrossLinkedList {
        private:
            int numRows;
            int numCols;
            vector<ListNode*> headRow; // 存储各行哨兵结点的数组
            vector<ListNode*> headCol; // 存储各列哨兵结点的数组
        
        public:
            // 构造函数：初始化哨兵结点
            // 假设矩阵下标从 1 开始，0 索引闲置或用作辅助
            CrossLinkedList(int m, int n) : numRows(m), numCols(n){
                headRow.resize(m + 1, nullptr);
                headCol.resize(n + 1, nullptr);

                // 初始化行的哨兵结点 (列坐标设为 -1 作为结束标志)
                for(int i = 1; i <= m; ++1){
                    headRow[i] = new ListNode(i, -1, 0, 0);
                    headRow[i]->left = headRow[i];
                    headRow[i]->up = nullptr;
                }

                // 初始化列的哨兵结点 (行坐标设为 -1 作为结束标志)
                for(int j = 1; j <= n; ++j){
                    headCol[j] = new ListNode(-1, j, 0, 0);
                    headCol[j]->up = headCol[j];
                    headCol[j]->left = nullptr;
                }
            }

            // 析构函数
            ~CrossLinkedList() {
                for(int i = 1; i <= numRows; ++i){
                    ListNode* p = headRow[i]->left;
                    while(p != headRow[i]){
                        ListNode* temp = p;
                        p = p->lesft;
                        delete temp;
                    }
                    delete headRow[i];
                }
                for(int j = 1; j <= numCols; ++j){
                    delete headCol[j];
                }
            }

            // 辅助函数：向十字链表中插入新结点 (维持 left 和 up 的降序排列)
            void insert(int r, int c, double v){
                if(v == 0) return;
                ListNode* newNode = new ListNode(r, c, v);

                // 插入到行循环链表 (按照 col 降序)
                ListNode* p = headRow[r];
                while(p->left != headRow[r] && p->left->col > c){
                    p = p->left;
                }
                newNode->left = p->left;
                p->left = newNode;

                // 插入到列循环链表 (按照 row 降序)
                ListNode* q = headCol[c];
                while (q->up != headCol[c] && q->up->row > r) {
                    q = q->up;
                } 
                newNode->up = q->up;
                q->up = newNode;
            }

            // 核心算法：主元操作  ----->   可以理解为高斯消元法中的核心一步
            // 参数 PIVOT 指向选定的主元结点
            void SP_Algorithm(ListNode* PIVOT) {
                if (PIVOT == nullptr) return;

                // SP1:[初始化，确定主行 I0, 主列 J0]
                int I0 = PIVOT->row;
                int J0 = PIVOT->col;
                double alpha = 1.0 / PIVOT->value;
                PIVOT->value = 1.0;

                ListNode* P0 = headRow[I0];
                ListNode* Q0 = headCol[J0];
                
                // 伪代码中的 PTR[1:n] 指针数组，用于记录各列当前处理到的垂直位置
                vector<ListNode*> PTR(numCols + 1, nullptr);

                // SP2: [处理主行 I0]
                P0 = P0->left;
                while (P0->col >= 0) { // col < 0 代表遇到了哨兵结点
                    int J = P0->col;
                    PTR[J] = headCol[J]; // PTR[J] 初始化指向列哨兵
                    P0->value = alpha * P0->value;
                    P0 = P0->left;
                }

                // SP3: [找新行 I, 并指定 P1]
                Q0 = Q0->up;
                while (Q0->row >= 0) { // 外层大循环：遍历主列 J0 上的非零元素
                    int I = Q0->row;
            
                    // IF I=I0 THEN GOTO SP3 (跳过主元所在的行)
                    if (I == I0) {
                        Q0 = Q0->up;
                        continue;
                    }

                    ListNode* P = headRow[I];
                    ListNode* P1 = P->left;
                    
                    // 重新将 P0 指向主行的第一个非零元素，准备开始 SP4
                    P0 = headRow[I0]->left; 

                    // SP4: [确定新列 J]
                    while (P0->col >= 0) { // 内层循环：遍历主行 I0 上的非零元素
                        int J = P0->col;
                        
                        // IF J=J0 THEN GOTO SP4 (跳过主元所在的列)
                        if (J == J0) {
                            P0 = P0->left;
                            continue;
                        }

                        // SP5: [P1 所指元素所在的列与 J 列比较]
                        while (P1->col > J) {
                            P = P1;
                            P1 = P1->left;
                        }

                        if (P1->col == J) {
                            // SP7:[主步骤操作 (结点已存在，执行更新)]
                            P1->value = P1->value - Q0->value * P0->value;
                    
                            // 浮点数判零，解决精度问题
                            if (abs(P1->value) < 1e-9) { 
                                // SP8: [删除零元素]
                                while (PTR[J]->up->row != P1->row) {
                                    PTR[J] = PTR[J]->up;
                                }
                                PTR[J]->up = P1->up;    // 垂直方向断开
                                P->left = P1->left;     // 水平方向断开
                                
                                ListNode* temp = P1;    // 对应伪代码的 AVAIL <- P1
                                P1 = P->left;
                                delete temp;            // 彻底销毁释放内存
                            } else {
                                PTR[J] = P1;
                                P = P1;
                                P1 = P1->left;
                            }
                        } else {
                            // SP6: [插入新元素 (结点不存在，创建并插入)]
                            while (PTR[J]->up->row > I) {
                                PTR[J] = PTR[J]->up;
                            }
                    
                            // 新建结点 X，计算其值
                            double newVal = -Q0->value * P0->value;
                            ListNode* X = new ListNode(I, J, newVal);
                    
                            // 改变水平链表指针
                            X->left = P1;
                            P->left = X;
                            P = X;
                            
                            // 改变垂直链表指针
                            X->up = PTR[J]->up;
                            PTR[J]->up = X;
                            PTR[J] = X;
                        }

                        // 继续处理主行的下一个元素
                        P0 = P0->left; 
                    } // END SP4 循环
            
                    // 当主行遍历完毕 (P0->col < 0) 触发伪代码的 IF J<0 THEN ...
                    Q0->value = -alpha * Q0->value;
            
                    // 准备进入下一行
                    Q0 = Q0->up; 
                } // END SP3 循环
            }

            // 辅助函数：打印矩阵结构，便于测试验证
            void printMatrix() {
                for (int i = 1; i <= numRows; ++i) {
                    ListNode* p = headRow[i]->left;
                    cout << "Row " << i << ": ";
                    if (p == headRow[i]) {
                        cout << "Empty";
                    } else {
                        while (p != headRow[i]) {
                            cout << "[" << p->row << "," << p->col << "]=" 
                                << fixed << setprecision(2) << p->value << " -> ";
                            p = p->left;
                        }
                        cout << "Sentinel";
                    }
                    cout << endl;
                }
            }

            // 辅助获取结点的指针，用于传入 SP_Algorithm
            ListNode* getNode(int r, int c) {
                ListNode* p = headRow[r]->left;
                while (p != headRow[r] && p->col > c) {
                    p = p->left;
                }
                if (p != headRow[r] && p->col == c) return p;
                return nullptr;
            }
        };
  ```
### 解释：
  - 要理解这个算法，我们需要分成两层来看：**第一层是它在数学上到底在算什么（What）**，**第二层是它在数据结构上是怎么操作的（How）**
#### 第一层：这玩意儿在数学上是干啥的？

这个算法的名字叫 **主元操作（Pivot Step）**。如果你学过**线性代数**，或者运筹学中的**单纯形法（Simplex Method）**，你对它绝对不陌生。

它的本质就是**矩阵的高斯消元法**中的核心一步。

假设我们解一个方程组（或者处理一个矩阵），我们选中了一个数字作为**主元（Pivot）**，接下来我们要完成两件事：
1. **把主元所在的那一行，整体除以主元的值。** （让主元变成 1）。
2. **用主元所在的行，去消灭掉主元所在列的其他数字。** （把主元正上方和正下方的数字变成 0，同时更新整个矩阵的其他数字）。

**举个通俗的例子：**
假设矩阵长这样，我们选 `2` 作为主元（第1行第1列）：
```text
[ 2   4 ]  <-- 主行
[ 3   8 ]  <-- 其他行
```
1. **处理主行**：主行全除以主元 `2`，变成 `[ 1, 2 ]`。
2. **处理其他行**：为了把第2行的 `3` 消掉，我们需要让：`第2行 = 第2行 - 3 * 新的主行`。
   所以第2行变成了：`[3 - 3*1, 8 - 3*2] = [0, 2]`。

这个十字链表的算法，核心就是在**稀疏矩阵**上完美地执行上述的数学运算！
*(注：伪代码中对主元所在列做了特殊处理 $Q_0 = -\alpha Q_0$，这是运筹学“单纯形表换基”的标准操作，把主列保存为负的乘子，便于后续计算，但整体逻辑就是高斯消元。)*

#### 第二层：算法一步步是怎么操作十字链表的？

在普通的二维数组 `int A[M][N]` 里，做上面的消元只需要两层 `for` 循环，非常简单。
**但为什么在这个十字链表里变得极其复杂？**
因为稀疏矩阵里**没有存 0**！如果计算后一个数字变成了 0，我们必须把它的内存**删掉（断开链表）**；如果计算后凭空产生了一个非零的数字，我们必须**新建一个节点插进去**！

我们一步步来拆解这台“手术”是怎么做的：

##### SP1: 【战前准备】
* **锁定目标**：找到传进来的主元节点（`PIVOT`），记录它的行号 $I_0$ 和列号 $J_0$。
* **计算乘子**：算出 $\alpha = 1 / 主元的值$。

##### SP2: 【处理主行】
* **扫荡主行**：顺着主元所在的行（$I_0$），向左遍历每一个非零节点 `P0`。
* **按比例缩小**：把主行上的所有数字，都乘以刚刚算出的 $\alpha$（就相当于除以主元）。

##### SP3: 【寻找需要做手术的行】
* **顺藤摸瓜**：顺着主元所在的列（$J_0$），向上寻找其他的非零节点 `Q0`。
* **为什么找它们？** 因为只有在主列上有非零数字的行，才需要参与矩阵消元。如果某行在这一列本来就是 0，那减去 0 还是它自己，就不需要动（稀疏矩阵的极大优势体现出来了！）。

##### SP4 & SP5: 【两行对比，准备做加减法】
现在我们拿到了两行数据：**主行 $I_0$**（被除过了） 和 **当前需要做手术的行 $I$**。
算法开始同时遍历这两行（这就是著名的**双指针归并思想**）：
* 看主行的列号 `J`，再去当前行找有没有相同的列号。

##### 此时会面临三种情况（这就是算法最复杂、最牛的地方）：

* **情况 A：对齐了！（对应 SP7）**
  当前行本来就有这个节点 `P1`，主行也有这个节点 `P0`。
  直接执行数学公式：`P1 的新值 = P1 原来的值 - Q0的值 * P0的值`。

* **情况 B：算完变成了 0！（对应 SP8 物理删除）**
  执行完情况 A 后，如果 `P1` 的新值恰好变成了 0（正负抵消了）。
  0 是不能留在稀疏矩阵里的！于是算法开始大动干戈：
  1. 把 `P1` 左边和右边的指针连起来（水平摘除）。
  2. 把 `P1` 上边和下边的指针连起来（垂直摘除）。
  3. `delete P1`，彻底销毁内存。

* **情况 C：无中生有！（对应 SP6 物理插入）**
  当前行在第 `J` 列本来没有节点（本来是 0），但是因为减去了主行对应的数字，这里凭空算出了一个非 0 的数值！
  1. `new ListNode` 诞生一个新节点 `X`。
  2. 计算它的值：`X 的值 = - Q0的值 * P0的值`。
  3. 小心地把 `X` 插入到当前行的水平链表，以及第 `J` 列的垂直链表中。

##### 最后收尾：
一层层循环执行下去，直到主列上的所有非零节点都被处理完，整个矩阵就完成了一次完美的高斯消元！

#### 总结
之所以这段代码看起来长得吓人，是因为它同时在干三件事：
1. **做数学运算**（矩阵高斯消元）。
2. **做链表的插入与删除**（水平方向维护单调性）。
3. **维护垂直方向的链表**（这也是为什么需要 `PTR` 数组，因为十字链表上下左右穿插，稍微断错一根线整个矩阵就废了）。

理解了这个，你就能体会到上世纪的计算机科学家在**内存极度匮乏**的年代，为了处理超大型稀疏矩阵（比如拥有 10万个方程，但大部分系数是0的方程组），设计出这套精妙绝伦的数据结构有多么了不起了~
