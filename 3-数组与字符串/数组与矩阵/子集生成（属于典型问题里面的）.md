## 题目说明
- 问题：输出一个集合的幂集
- 参考题目：[洛谷 B3622 枚举子集（递归实现指数型枚举）](https://www.luogu.com.cn/problem/B3622)

### 代码实现
```cpp {.line-numbers}
const int maxsize = 10;
// 若开头是 0，则开头变为 1
// 若开头是 1，则把从开头一直连着的 1 全变为 0，然后把后面的一个 0 变为 1
//二进制数加计数器
void AddOne(int a[], int n){
    int i = 0;
    while(a[i] != 0)
        a[i++] = 0;
    a[i] = 1;
}
// 打印对应值为 1 的那些元素
void PowerSet(char s[], int n){
    int a[maxsize] = { 0 };
    while(a[n] == 0){
        printf("{ ");
        for(int i = 0; i < n; i++)
            if(a[i] == 1)
                printf("%c ", s[i]);
        printf("}\n");
        AddOne(a, n);
    }
}

int main(){
    char s[maxsize] = {'a','b','c','d','e','f','g'};
    int n = 4;
    PowerSet(s, n);
    return 0;
}
```

### 拓展
- 题目说明：请编写程序以**字典序**输出一个集合的所有子集。输入为一个正整数 `n（n ≤ 15）` 表示集合包含的元素个数。约定元素的编号为 `a, b, c, d …` 。若 `n=1` ，集合元素为 `a` ；若 `n=3` ，集合元素为 `a, b, c` ；若 `n=4` ，集合元素为 `a, b, c, d` ；以此类推。
- 示例：
  - `n = 3` 输出：`{ } { a } { a , b } { a , b , c } { a , c } {b} { b,c } { c }`
  - `n = 4` 输出：`{ } { a } { a , b } { a , b , c } { a , b , c , d } { a , b , d } { a , c } { a , c , d } { a , d } {b} { b,c } { b , c , d } { b,d } { c } { c , d } {d}`
- 代码实现：
    ```cpp {.line-numbers}
    #include <iostream>
    #include <vector>

    using namespace std;

    // 回溯函数
    // cur_set: 当前正在构建的子集
    // start_idx: 当前可以加入子集的字母的起始偏移量 (0代表'a', 1代表'b'...)
    // n: 集合中元素的总个数
    void dfs(vector<char>& cur_set, int start_idx, int n){
        // 1. 每次进入函数，代表生成了一个合法的子集，先将其按格式打印出来
        cout << "{";
        for(int i = 0; i < cur_set.size(); ++i){
            cout << cur_set[i];
            // 最后一个元素后面不加逗号
            if(i < cur_set.size() - 1){
                cout << ",";
            }
        }
        cout << "}\n";

        // 2. 尝试向当前子集中添加后续的元素
        for(int i = start_idx; i < n; ++i){
            // 把第 i 个字母加入子集
            cur_set.push_back('a' + i);
            // 递归进入下一层，注意下一层只能从 i + 1 开始挑字母，保证字典序和不重复
            dfs(cur_set, i + 1, n);
            // 回溯：把刚才加进去的字母退出来，尝试下一个字母
            cur_set.pop_back();
        }
    }

    int main() {
        ios_base::sync_with_stdio(false);
        cin.tie(NULL);

        int n;
        if (cin >> n) {
            vector<char> cur_set; // 用来存放当前子集的容器
            dfs(cur_set, 0, n);   // 从第 0 个字母 ('a') 开始搜索
        }

        return 0;
    }
    ```
